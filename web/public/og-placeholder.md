# Open Graph Image Placeholder

This is a placeholder for the default Open Graph image.

## Required Specifications

- **Filename**: og.png
- **Dimensions**: 1200x630px (Facebook/Twitter recommended)
- **Format**: PNG or JPG
- **File Size**: < 1MB
- **Usage**:
  - Default OG image when profile has no image
  - Social media sharing fallback
  - Link previews

## Design Guidelines

- Include site branding (logo + name)
- Clear, readable text
- High contrast
- Professional appearance
- Works well as thumbnail
